# AUTO RENTAL WEB APPLICATION 🚗

A 3rd-year internship project built with PHP and MySQL that allows users to search and book cars using various filters, manage their profile, and download booking receipts. Rental companies can manage their car inventory and view all bookings through the admin panel.

## FEATURES ✨

- **Car Search** — Search for available cars by manufacturer, model, and price range.
- **Booking System** — Book cars for specific dates with automatic price calculation.
- **Receipt Download** — Download a PDF receipt for each completed booking.
- **User Authentication** — Register, log in, and reset password via email verification.
- **Admin Panel** — Manage cars and view all bookings from a dedicated backend interface.

## TECHNOLOGIES 🚀

- **PHP** — Server-side scripting language for backend development.
- **MySQL** — Relational database for storing application data.
- **HTML/CSS/JavaScript** — Frontend technologies for building the user interface.
- **Bootstrap** — Frontend framework for responsive, mobile-first design.

## INSTALLATION ⚙️

> This guide assumes you have a local server environment such as XAMPP, WAMP, or MAMP installed.

1. Clone the repository to your local machine.
2. Import `aimanecars.sql` into your MySQL database to create the schema and load sample data.
3. Configure the database connection settings in `partials/connection.php`.
4. Start the local server environment and open the application in your browser.
5. The default admin password can be found in `aimanecars.sql` under the `admin` table.
